# Dev Tools

A collection of small tools I build to scratch my own itches while working —
mostly around developer workflow and AI tooling. Each tool lives in its own
subfolder and is self-contained.

## Tools

| Tool | What it does |
|------|--------------|
| [**cc-token-dashboard**](./cc-token-dashboard) | A macOS menu bar app that shows your Claude Code token usage in real time, by parsing local `~/.claude/projects` transcripts. No API, no network. |

_More to come._

## Using a tool

Each subfolder has its own README with build/run instructions. They're independent —
clone the repo and build whichever one you need.
